# AIClient2API

搜索目标：`title="AIClient2API - 管理控制台"`

使用默认密码登录后，下载 ZIP 配置包并在内存中解压，提取 antigravity 和 kiro 两类凭证。

## 运行

```bash
bun run aiclient2api
```

## 输出

- `output/aiclient2api-logins.json` — 登录成功的站点
- `output/aiclient2api-antigravity.json` — Antigravity 凭证
- `output/aiclient2api-kiro.json` — Kiro 凭证
